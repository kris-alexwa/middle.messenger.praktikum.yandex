- SCSS/SASS

- Parcel
- Express
- TypeScript

## **Установка и запуск**
Установка зависимостей проекта:
``` 
npm install
```
Локальный запуск проекта
``` 
npm run start
```
Сборка проекта
``` 
npm run build
```
